# 🏥 HealthAppoint App

HealthAppoint is a **React Native (Expo)** based healthcare appointment booking application developed as a **college assignment project**.  
The app allows users to book doctor appointments, analyze symptoms using AI, and get doctor suggestions intelligently.

# 🚀 Tech Stack

## Frontend
- **React Native**
- **Expo**
- JavaScript
- React Navigation
- AsyncStorage

## Backend
- **Node.js**
- **Express.js**
- **MongoDB**

## Additional Services & APIs
- **Groq API** → AI-based symptom analysis and doctor suggestions
- **Cloudinary** → Medical image upload and storage
- **Socket.io** → Real-time slot booking and appointment updates

---

# ✨ Features

## 👨‍⚕️ Doctor Appointment Booking
- Browse available doctors
- View doctor details
- Select appointment date and time slot
- Confirm booking instantly

---

## 🤖 AI Symptom Analysis
Users can enter their symptoms and the app uses **Groq AI API** to:

- Analyze symptoms
- Suggest possible health issues
- Recommend suitable doctor specialization

---

## ⚡ Real-Time Slot Booking
Using **Socket.io**:

- Live appointment slot updates
- Prevents double booking
- Instant synchronization between users

---

## 🖼️ Image Upload Support
Using **Cloudinary**:

- Upload medical reports/images
- Secure cloud storage
- Fast image retrieval

---

## 🔐 Authentication
- User signup/login
- Secure session handling

---

# 🛠️ Setup Instructions

## Frontend Setup

```bash
git clone https://github.com/yogeshkumawat2027/HealthAppointApp
cd HealthAppointApp
npm install
npx expo start
```

## Backend Setup

```bash
git clone https://github.com/yogeshkumawat2027/healthappoint-backend-v2
cd healthappoint-backend-v2
npm install
npm run dev
```
